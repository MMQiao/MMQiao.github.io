Python Learning

22 May 2023

python的单行注释，默认#和注释之间要有一个空格
# 注释

多行注释支持换行，三引号中间的内容都是注释
“““注释”””

type()函数可以查看数据类型，其返回值也可以赋值给变量
需要注意的是，python的变量是没有类型的，type()返回的只是其中存储数据的类型

print()函数可以输出多个数据，数据直接用逗号,隔开
print()函数会自动换行，如果需要不换行，需要在括号内加上end=’ ’
例:print(“hello”,end=’ ‘)

python中整数、浮点数都能转成字符串，但如果要把字符串转化成整数，要保证字符串中是正确的数字

标识符是用户在编程的时候所使用的一系列名字，用于给变量、类、方法命名。
标识符命名的规则有三点：
1.标识符命名中只允许出现：
·英文
·中文
·数字
·下划线
#不推荐使用中文，不可以使用数字开头
2.大小写敏感
3.不可以使用关键字作为标识符
python中有一系列单词称为关键字，有特定用途

变量命名规范：
1.见名知意
2.下划线命名法，不同单词之间用下划线划分
3.英语字母全小写

python算术运算符：
+相加
-相减
*相乘
/除以
//整除
**次方
=赋值
复合运算符，例：+=

字符串定义法：
单引号定义法、双引号定义法、三引号定义法
引号的嵌套：
1.可以使用\来进行转义 2.可以使用单引号包括双引号或者双引号包括单引号
字符串可以使用+进行拼接，但无法和其他类型数据拼接
字符串格式化：
1.在字符串中使用%s进行占位，随后加上 %变量 进行补全
多个%s进行占位时，后面应该使用 %(变量，变量，…)的格式进行补全
字符串格式化中，补全的变量不必是字符串，可以是整数、浮点数
#整数的占位符是%d，浮点数的占位符是%f，占位符会把对应的变量转化为对应数据类型

可以使用辅助符号"m.n"来控制数据的宽度和精度
·m控制宽度，要求是数字，如果设置的宽度小于数字本身，不生效
·.n控制小数点精度，要求是数字，同时会进行小数的四舍五入
小数点和小数部分也算入宽度计算

2.f"内容{占位}“进行快速格式化
这种方法不限制数据类型，但也不控制精度

input函数接收的数据都是字符串格式，需要手动转换

if条件语句的使用：
if 要判断的条件:
条件成立要做的事
elif 条件2:
条件2成立要做的事
else:
所有条件不满足时做的事
要注意缩进

while循环语句的使用：
while 条件:
条件满足时要做的事
条件的变化（保证循环能正常结束）
条件必须是布尔值

for循环语句的使用：
for 临时变量 in 待处理数据集（序列）: 循环满足条件时执行的代码
for循环也被称为遍历循环，无法定义循环条件
如果想要在for循环外访问临时变量，应该在for循环之前提前定义该变量，而不是直接访问（可以访问，但不符合规范）

range()函数的三种用法
1.range(num)获得一个从0开始的长度为num的序列
2.range(num1,num2)获得一个从num1开始，num2-1结束的序列
3.range(num1,num2,step)获得一个从num1开始，num2-1结束，数字之间步长为step的序列

continue语句：
忽略剩下的语句，继续执行下一次循环

break语句：
忽略剩下的语句，结束当前循环
