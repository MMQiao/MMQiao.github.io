Ubuntu Foundation

22 May 2023

scp命令，意为secure copy，可以在本地电脑和服务器之间传递文件
命令格式为
scp -P xx user@remote:file_address local_file_address
可以用-r选项来传递文件夹

在服务器端使用exit命令中断连接

服务器可以设置免密码登录，以及设置别名方便登录

chmod修改权限
r代表读权限，w代表写权限，x代表执行权限，s代表
chmod -R 755 文件名|目录
三个数字分别代表拥有者、组、其他用户的权限
r权重为4，w权重为2，x权重为1

groupadd添加组
groupdel删除组
chgrp修改文件或目录所属组，修改目录时需要加上-R选项递归

useradd新建用户-m选项表示建立家目录，-g选项表示用户所属的组
userdel删除用户
passwd修改用户密码

id命令可以查看用户id和组id等信息

who命令可以查看当前登录到终端的用户信息
whoami可以查看自己登录的用户信息

usermod可以设置主组和附加组
也可以指定用户登录的shell

which命令，查询终端命令保存在系统中的位置

su切换用户
不加用户名则切换到root
切换其他用户时，加上-可以切换到其他用户家目录
使用exit退出

chown修改目录|文件拥有者

date查看系统时间
cal查看日历

df显示磁盘剩余空间disk free
du显示目录下的文件大小
-h选项，使用人性化的方式显示

ps aux查看进程的详细状况process status，不需要使用-aux
a显示终端上所有进程，包括其他用户的进程
u显示进程的详细状态
x显示没有控制终端的进程

top动态显示进程并排序

kill终止进程
kill -9 进程号，-9选项表示强行终止

find在目录下查找符合条件的文件
find [目录] -name “file”

ln命令创建软链接
ln -s 被链接的源文件 链接文件
没有-s选项建立的是硬链接
建立软链接时，对于源文件，最好使用绝对路径

对于软链接，其指向的是对应文件的文件名，当删除源文件时，其文件名消失，软链接失效
对于硬链接，其本质是文件的别名，当删除源文件的本名时，对应的数据不会消失

#打包文件
tar -cvf 打包文件.tar 被打包的文件/路径
#解压文件
tar -xvf 打包文件.tar
-c生成档案文件，创建打包文件
-x解开档案文件
-v列出归档解档的详细过场，显示进度
-f指定档案文件名称，f后面一定是.tar文件，所以必须放选项最后

tar命令只负责打包，不负责压缩
-z选项可以调用gzip命令，对文件进行压缩或解压,格式为tar.gz
-j选项可以调用bzip2命令，对文件进行压缩或者解压，格式为tar.bz2
-C选项可以指定解压缩的目录，解压缩的目录必须存在

#安装软件
sudo apt install 软件包
#卸载软件
sudo apt remove 软件包
#更新软件
sudo apt update && sudo apt upgrade
